# Hit Visuals Download Page

This folder is a ready-to-host static site.

## Files

- `index.html` - landing page
- `styles.css` - styling
- `site.js` - small UX helpers
- `files/hitvisuals-1.0.0.jar` - the downloadable mod jar

## Hosting

Any static hosting works (GitHub Pages, Cloudflare Pages, Netlify, etc.).

For GitHub Pages, a common setup is:

1. Repository settings -> Pages
2. Source: `Deploy from a branch`
3. Branch: `main`
4. Folder: `/docs`
